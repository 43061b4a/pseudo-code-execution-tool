Variable Already Defined

Every variable that is defined must have a different name. It does not matter if one variable type is int and the other is array, all variables must have different names. It is good practice to name your variables with intuative names, like 'weight' or 'height', or 'time_array'.

Code:

int a, a, a
# This will cause an error as a was already defined

array a[10]
# This will cause an error as a was already defined as an int

define main {


}
