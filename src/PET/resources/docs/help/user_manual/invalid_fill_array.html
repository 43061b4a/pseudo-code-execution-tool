Invalid FillArray Line

This error occurs when you are trying to use the built in function 'FillArray' in an invalid way. See below for examples of proper syntax.


array a[10]
bool b

define main {

 # FillArray must be passed a valid array
 # This is okay
 FillArray a
 FillArray a    # comment after line
 
 # This will cause this error
 FillArray 12
 FillArray a extra stuff
 

}
