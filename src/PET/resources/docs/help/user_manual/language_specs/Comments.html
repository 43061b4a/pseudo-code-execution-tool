Comments

Comments can be inserted into your code to help others and yourself to understand what you are doing when reading the code. Comments are indicated using the # symbol. Comments can appear anywhere and take up the rest of the line that they are on.

Code:

# a comment can be a line of its own
int i  # or can be on the same line as real code
define main {
    #however, real code can not come on a line after a comment

}
