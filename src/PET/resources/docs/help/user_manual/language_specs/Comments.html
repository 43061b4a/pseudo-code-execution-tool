Comments

Comments can be inserted into your code to help others and yourself to understand what you are doing when reading the code. Comments are indicated using the # symbol. A comment can be on a line of its own or can appear after a line containing code. Any text following a # symbol will be ignored by the compiler.

Code:

# a comment can be a line of its own
int i  # or can be on the same line as real code
define main {
    #however, real code can not come on a line after a comment

}
