Write Command

The Write and WriteLine commands are used to print to the terminal. Write takes space separated arguments. It will append a space to the start of a variables value when printing it. The difference between Write and WriteLine is that WriteLine will write a newline character at after all of its arguments. This will force the content of the next Write command to appear on a new line. WriteLine can also be called with no arguments in which case it will only print the new line character.

Code:

int i
define main {

    Write "some text and a variable" i
    WriteLine "this will be on the same line as the above command"
    Write "this will be on a new line"

}
