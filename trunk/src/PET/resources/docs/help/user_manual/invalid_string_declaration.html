Invalid string declaration

This error occurs when you try to declare a string in an invalid way.

Code:

# this is okay
string a, b, c
string my_string
string my_other_string = "some text"
string another_string = my_other_string
# this is not
string my_string = 10
string my_string = true
string my_string = "remember you can not have a " inside a string"

define main {
