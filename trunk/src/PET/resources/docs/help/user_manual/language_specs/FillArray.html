fillarray

fillarray is a nice way of filling an array with random numbers. This is especially useful for initializing an array before sorting the numbers in it. This command takes one argument which must be a valid name of an array that is already defined with a size. After execution the array will be filled randomly with numbers greater than 0 and less than or equal to 10 times the size of the array.

Code:

array my_array[10]
define main {

    fillarray my_array
    # my_array will now contain random numbers between 1 and 100
    
}
