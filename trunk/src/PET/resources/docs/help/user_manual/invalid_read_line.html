Invalid read line

This error occurs when you have an invalid read line. See example below for the proper syntax. Remember you can only read into one variable per line.

Code:

                
string my_string
bool my_bool
int my_int
array my_array[10]

define main {

read my_string
read my_bool
read my_int
read my_array[0]

write "You entered" my_string ", " my_bool ", " my_int ", " my_array[0] "."

}
